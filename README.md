# Study Buddy & Productivity Dashboard

A web-based productivity application designed to help students manage tasks, stay organized, and improve focus during study sessions.

---

## Overview

The Study Buddy Dashboard combines task management, time tracking, and resource organization into a single, user-friendly interface. It is built to support students in maintaining consistent study habits and improving productivity.

---

## Features

 ### Task Management

 * Create, update, and delete tasks
 * Organize tasks by category (e.g., Math, Science, Coding)
 * Mark tasks as completed
 * Track progress across tasks

---

 ### Pomodoro Timer

 * Integrated focus timer based on the Pomodoro technique
 * Supports structured study sessions with break intervals
 * Helps improve concentration and time management

---

 ### Resource Hub

 * Store useful links and study materials
 * Add personal notes for quick reference
 * Centralized access to learning resources

---

 ### Data Persistence

 * Uses browser Local Storage to save tasks and data
 * Ensures information remains available after page reload

---

 ### Optional Enhancements

 * Daily motivational quotes via API
 * Weather updates integration
 * Theme customization (light/dark mode)

---

 ## Technologies Used

* HTML5

* JavaScript (Vanilla)
* Local Storage API
